✝️ Catholic Bible

Offline Douay-Rheims Bible for Flipper Zero — full Catholic canon, bookmarks, history, Missal & Guides.

What it is

A fully offline app that turns your Flipper Zero into a portable Scripture reader. No account, no network — just the text. The full Bible (73 books, 34,827 verses) is bundled in the FAP; no SD card required to read.

Features

  • Browse — Book → Chapter → Verse navigation (all 73 books)
  • Last read — Jump back to your last verse
  • Bookmarks — Mark verses; list and open from the menu
  • History — Recent verses; tap to reopen
  • Missal — Today’s Mass, liturgical calendar, Mass prayers & responses, readings (when missal.bin is present)
  • Prayers — Common prayers from devotional.bin
  • Guides — Order of Mass, OCIA, Lenten, Easter, Pentecost, Sacraments, Marrying Catholic (scaffold)
  • Rosary & Confession — Menus in place; content via build assets

Install

Install the Catholic Bible FAP from the Flipper app catalog (or build from source with ufbt build). With Flipper connected: ufbt launch to deploy. Optional: copy assets to SD /apps_data/bible/ to override bundled files.

Data files & SD card setup

All data files go in one folder on your Flipper’s microSD card: apps_data/bible. If you connect the Flipper via USB, open qFlipper and use the file browser to go to the SD card; create the folder apps_data, then inside it create bible. Or remove the SD card, put it in your computer’s card reader, and create apps_data/bible in the root of the SD. Put every file listed below directly inside apps_data/bible (and for search, put the search_shards folder there too).

Install everything (one download)

Download everything Repository ZIP (main branch) — contains the files folder with all data files.

Install everything:

  1. Download the ZIP from the link above and unzip it on your computer.
  2. On your Flipper’s SD card, create the folder apps_data/bible (use qFlipper’s file browser with the Flipper connected, or put the SD in your computer and create the folder there).
  3. Copy the entire contents of the unzipped repo’s files folder into apps_data/bible/. That means all .bin files and the search_shards folder. You should end up with apps_data/bible/bible_text.bin, apps_data/bible/verse_index.bin, apps_data/bible/missal.bin, apps_data/bible/devotional.bin, apps_data/bible/rosary.bin, apps_data/bible/confession.bin, apps_data/bible/search_shard_map.bin, and apps_data/bible/search_shards/ (with all the shard files inside).
  4. Safely eject the SD card (or disconnect the Flipper). Open the Catholic Bible app on the Flipper; all features will use the data you copied.

verse_index.bin

What you need: bible_text.bin, verse_index.bin. The FAP already includes the full Bible; you only need these if you want to override with your own build or restore files.

  1. Download the two files above (or build from source: python3 tools/build_bible_assets.py, then copy from dist/apps_data/bible/).

Missal (today’s Mass, calendar, prayers, readings)

Download missal.bin

What you need: missal.bin. Without it, the Missal menu shows “data not loaded.”

  1. Download missal.bin from the link above.
  2. Copy it into apps_data/bible/ on the SD card.
  3. Optional: to rebuild from source, edit assets/source/missal.json, run python3 tools/build_missal.py, then copy files/missal.bin to the SD.

Download search_shard_map.bin
search_shards folder (open folder, download each shard_XXX.bin or clone the repo and copy the folder)
